# 🎬 IMDB World Wide Project  
### Avaliação de Filmes e Cinema Mundial

Este projeto foi desenvolvido como parte da disciplina de **Business Intelligence II**, com o objetivo de explorar, modelar e visualizar dados do universo cinematográfico global, utilizando como base informações inspiradas no IMDB.

O resultado é um dashboard interativo que permite analisar desempenho financeiro, popularidade e características da indústria do cinema ao longo do tempo.

---

## 🚀 Objetivos do Projeto

- Analisar dados de filmes em escala global  
- Explorar métricas de arrecadação, investimento e lucro  
- Identificar padrões por estúdios, gêneros e títulos  
- Desenvolver um modelo semântico eficiente  
- Criar um dashboard visual e intuitivo no Power BI  

---

## 🧠 Tecnologias e Ferramentas

- **Power BI** (Modelagem e Visualização)
- **Power Query** (ETL)
- **DAX** (Métricas e cálculos)
- **Modelagem Dimensional**
- **Excel / CSV** (Fonte de dados)

---

## 📊 Principais Análises

O dashboard desenvolvido permite explorar:

### 💰 Indicadores Gerais
- Ticket médio de arrecadação  
- Arrecadação total global  

### 🎥 Produção por Estúdios
- Distribuição de filmes por grandes produtoras  
- Comparativo entre estúdios como:
  - Universal Pictures  
  - Warner Bros  
  - Paramount Pictures  

### 📈 Desempenho Financeiro
- Comparação entre:
  - Arrecadação  
  - Investimento  
  - Lucro  
- Evolução por título  

### 🏆 Top Filmes
- Ranking dos filmes com maior arrecadação  
- Destaques como:
  - Avatar  
  - Avengers  
  - Titanic  

### 🎭 Gêneros Cinematográficos
- Distribuição de nomeações por gênero  
- Identificação dos gêneros mais populares
